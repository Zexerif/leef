Persistent Local Memory & Bookmarks!

  • Privacy Policy Upgrade: We've successfully disabled the strict "in-memory-only" restriction to allow core browser functionalities like Cache, IndexedDB, LocalStorage, and explicitly Bookmarks to securely persist across reboots onto your local storage.
  • Bookmarks System: Added a gorgeous new "Add to Bookmarks" dropdown menu tied to the Star icon, which successfully grabs site context, pulls Google Favicons, and manages routing.
April 16, 2026

Dynamic RSS News Feed

  • Live News Delivery: Using internal HTTPS routing inside Electron's Node layer to securely intercept Yahoo RSS feeds without triggering CORS flags.
  • Style Fixes: Fixed alignment padding around the News module containers, resolving broken interactive layouts alongside restoring classic OS typography specifically to the underlying Leef Home Dashboard layout.

Initial Alpha Build
